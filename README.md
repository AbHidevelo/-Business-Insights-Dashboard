# -Business-Insights-Dashboard
This project is an end-to-end data analysis dashboard built using Python, PostgreSQL, and Streamlit.  It analyzes retail sales data to uncover insights across regions, categories, and time.
🛠 **Tech Stack**
Python (Pandas)
SQL (PostgreSQL)
Streamlit
Data Analysis
📈** Features**
KPI metrics (Sales, Profit, Orders)
Monthly sales trends
Product performance analysis
Region & category insights
Interactive dashboard
